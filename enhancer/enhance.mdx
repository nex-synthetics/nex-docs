NEX comes with a built in Image Enhancer that upscales the image to 2x its original size as well as reimagines the image. 

## How to use the Image Enhancer


#### 1. On the Console, click on the Image Enhancer, or head there directly https://console.nex.art/enhance.

  <img src="/images/enhance/1.png" style={{ borderRadius: '0.5rem' }} />
</Frame>

#### 2. Select an image to enhance.

<Frame>
  <img src="/images/enhance/2.png" style={{ borderRadius: '0.5rem' }} />
</Frame>

#### 3. Describe the ideal output as well as input image in the Prompt box.

<Frame>
  <img src="/images/enhance/3.png" style={{ borderRadius: '0.5rem' }} />
</Frame>
<Frame>
  <img src="/images/enhance/4.png" style={{ borderRadius: '0.5rem' }} />
</Frame>


#### 4. Adjust the Creativity slider. A higher creativity results in a greater amount of changes and a lower creativity effects the image composition minimally. 

<Frame>
  <img src="/images/enhance/5.png" style={{ borderRadius: '0.5rem' }} />
</Frame>

### Creativity Slider
A creativity scale of < 0.4 is recommended. Depending on the input image quality and resolution, enhancing an image with a creativity scale > 0.4 can result in an increased amount of unwanted artifacts and distortion.
